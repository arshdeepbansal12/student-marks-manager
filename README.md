# Student Marks Manager

A simple Python utility to collect student details (Name, Roll Number, and Marks) from user input and save them into a structured text file. It automatically calculates and appends the highest, lowest, and average marks to the file.

## Features
- Interactive console input for multiple students.
- Automated file writing to `marks.txt`.
- Data processing to find class performance statistics.

## How to use
1. Ensure you have Python installed.
2. Clone this repository:
   ```bash
   git clone [https://github.com/arshdeepbansal12/student-marks-manager.git](https://github.com/arshdeepbansal/student-marks-manager.git)

   Run the script:

Bash
python main.py

Follow the on-screen prompts to enter student details.

The results will be saved in marks.txt within the same directory.
